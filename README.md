# CloudCast ⛅

A premium, responsive web-based weather dashboard application that provides real-time atmospheric updates using live API integration. The interface features a clean glassmorphism card design optimized for seamless user focus.

## Features

- **Search Weather by City:** Input field to dynamically fetch location-specific live data.
- **Display Current Temperature:** Instantly renders precise temperature values.
- **Display Humidity:** Returns active humidity metrics in percentage format.
- **Display Wind Speed:** Provides accurate wind speed metrics scaled in meters per second ($m/s$).
- **Unit Conversion Toggle:** Easily switch temperature units between Celsius ($^\circ\text{C}$) and Fahrenheit ($^\circ\text{F}$).
- **Clean and User-Friendly Interface:** Styled with a vibrant modern sky gradient and premium frosted glass aesthetic.

## Technologies Used

- **HTML5:** Structured semantic configuration.
- **CSS3:** Advanced fluid typography, custom linear gradients, and glassmorphism interface panels.
- **JavaScript (ES6):** Asynchronous fetch requests with client-side fallback security loops and dynamic DOM handling.
- **OpenWeatherMap API:** Primary data pipeline synchronization handler.
